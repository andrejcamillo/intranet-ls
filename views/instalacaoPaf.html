Instalação e Configuração do PAF



  • Acessar o endereço \\10.0.0.201\Publico\Suporte\PAF.
  • Transferir o arquivo PAF.rar para o computador do cliente, criar uma pasta e extrair dentro dessa pasta.

  • Executar o instalador Firebird.exe. Não precisa alterar nada, somente ir clicando em avançar até concluir a instalação.

  • Executar o instalador LS PAFECF Setup.exe. Só ir avançando e no final, DESMARCAR na última tela a opção "Executar".
  • Após clicar em concluir.

  • Acessar a pasta \\10.0.0.201\Publico\Suporte\PAF\Config. Copiar o arquivo LS PAF Config.exe e colar na pasta de instalação do PAF.

  • Executar o LS PAF Config.exe, ir na aba ECF Autorizados para funcionamento.

  • Selecionar o modelo da impressora e a porta COM que ela está conectada, após isso clicar em Capturar.

  • Será preenchido os campos da parte de sistema, anote esses dados pois será utilizado posteriormente.

  • Para preencher o CNIEE é necessário acessar o caminho \\10.0.0.201\Publico\Suporte\PAF e abrir o arquivo CNIEE.pdf.

  • Dentro do pdf busque o modelo da impressora e copie o CNIEE para preencher no LS PAF Config.

  • Após preencher as informações clicar em Adicionar ECF. Após isso clicar em Salvar Arquivo Auxiliar.

  • Ao terminal a configuração da impressora é necessário que seja excluído o arquivo LS PAF Config.exe da pasta do PAF.

  • Acessar o sistema LS, ir em Configurações > Sistema > Impressora Fiscal > Novo e preenhcer os campos conforme dados da impressora.

  • Acessar o endereço C:\Program Files (x86)\LS PAFECF, abrir o arquivo LS_PAF_Config.ini e preencher com as seguintes informações:

    1. ERPHost = Endereço do servidor onde está a base do PostgreSQL.

    2. ERPBase = Nome da base de dados do PostgreSQL.

    3. ERPPort = Porta do PostgreSQL no servidor.

    4. Filial = Número da filial no sistema LS.

    5. IdECF = Código da Impressora Fiscal cadastrada no sistema LS.

    6. NFeSerie = Série de emissão de NFe do sistema LS + 1.

  • Executar o instalador LS PAFECF Servico Setup.exe. Não precisa alterar nada, somente ir clicando em avançar até concluir a instalação.

  • Acessar o endereço C:\Program Files (x86)\LS PAFECF Srv, abrir o arquivo LS_PAF_Config.Ini e preencher com as seguintes informações:

    1. ERPHost = Endereço do servidor onde está a base do PostgreSQL.

    2. ERPBase = Nome da base de dados do PostgreSQL.

    3. ERPPort = Porta do PostgreSQL no servidor.

    4. PAFBase = Endereço completo da base do PAF da instalação do LS PAFECF. Ex.: C:\Program Files (x86)\LS PAFECF\LS_PAFECF.FDB.

    5. Filial = Número da filial no sistema LS.

    6. IdECF = Código da Impressora Fiscal cadastrada no sistema LS.

    7. NFeSerie = Série de emissão de NFe do sistema LS + 1.

  • Buscar no menu iniciar o CMD, executar como Administrador, acessar a pasta da PAF através do comando "cd caminho_da_pasta"

  • Instalar o serviço LS_PAF_Srv através do comando "LS_PAF_Srv.exe /install".

  • Aperte Ctrl + R e execute o comando services.msc, localize o serviço LS_PAF_ECF e clique em Iniciar.

  • Execute o arquivo LS_PAFECF.exe e analise se irá executar corretamente.

  • Para trocar a logo é necessário colocar a logo da empresa na pasta do PAF no formato .png para que o sistema reconheça.
